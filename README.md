### Setup

##### Main Repository
1. `git clone git@github.com:nattayamairittha/webapi_unity3d.git`
2. `cd webapi_unity3d`
3. `copy a directory WebAPI Client to Unity3D Engine`
4. `npm i`
5. `node index.js`
